# A Local Kids Toy Store Platform

A modern and user-friendly e-commerce platform for toys. It is fully responsive and designed to provide a smooth shopping experience. 



## Project Overview

ToyStore is a fully functional e-commerce solution designed to manage online sales of toys. The platform efficiently handles product display, user authentication, shopping cart management, and the checkout process.
## Key Features

* **Secure Checkout System:** Implemented with Stripe/PayPal integration for safe and seamless payment processing.
* **User Authentication (JWT):** Secure registration and login functionality with protected user profiles.


## Tech Stack

- **React.js** : For component-based architecture and state management. 
- **Tailwind CSS** : For fast and responsive styling with a utility-first approach. 
- **JavaScript (ES6+)** : For application logic and dynamic interfaces. 


## Dependencies

* `axios`
* `react-router`
* `react-toastify`
* `swiper`
* `tailwindcss`
* `react-icons`
* `firebase`
## Installation

Clone the repo and install dependencies

```bash
  git clone https://github.com/emon-ahmed-60/Toy-store.git
  cd Toy-store
  npm install
```

Set up environment variables by creating a .env file in the root directory:

```bash
  VITE_apiKey=Your_apikey
  VITE_authDomain=Yout_authDomain
  VITE_projectId=Your_projectId
  VITE_storageBucket=Your_storageBucket
  VITE_messagingSenderId=Your_messagingSenderId
  VITE_appId=Your_appId
```
Run the application:

```bash
  npm run dev
```
